# Full Stack Development (FSD) Lab
This repository contains my Full Stack Development (FSD) lab assignments, mini projects, and practical implementations completed during my B.E. Computer Science Engineering coursework.

## 📚 Repository Contents

### React Mini Projects

The `react-mini-projects` folder contains beginner-friendly React applications built using **React** and **Vite**.

| Project | Description | Concepts |
|---------|-------------|----------|
| Counter App | Increment, decrement, and reset a counter | Components, useState, Event Handling |
| Stopwatch App | Stopwatch with start, stop, and reset functionality | useState, useEffect, setInterval |
| Todo List App | Task management application | State Management, Lists, Props, Events |

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/Imdilshan/24bcs10854_Md_Dilshan_FSD.git
```

Navigate to any project:

```bash
cd react-mini-projects/counter-app
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
