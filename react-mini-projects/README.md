# React Mini Projects

A collection of beginner React projects built using React + Vite.

## Projects

- Counter App
- Stopwatch App
- Todo List App

## Tech Stack

- React
- Vite
- JavaScript
- CSS

## Getting Started

Clone the repository

```bash
git clone https://github.com/imdilshan/react-mini-projects.git
```

Go into any project

```bash
cd counter-app
npm install
npm run dev
```
